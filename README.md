## 📂 Problems Covered

### 1️⃣ Two Sum (Two Pointer Approach)

**File:** `two_sum_two_pointer.py`

🔍 Find two numbers in an array that add up to a target value.

🔗 LeetCode: https://leetcode.com/problems/two-sum/

⚙️ Approach:

* Sort the array
* Use two pointers (left & right)
* Move pointers based on sum

⏱️ Time Complexity: `O(n log n)`
💾 Space Complexity: `O(1)`

⚠️ Note: This approach does not preserve original indices.

---

### 2️⃣ Squares of Sorted Array

**File:** `sorted_squares.py`

🔍 Given a sorted array, return a new array of squares in sorted order.

🔗 LeetCode: https://leetcode.com/problems/squares-of-a-sorted-array/

⚙️ Approach:

* Separate negative and positive numbers
* Square all elements
* Reverse the negative part
* Merge both sorted arrays

⏱️ Time Complexity: `O(n)`
💾 Space Complexity: `O(n)`

---

### 3️⃣ Merge Two Sorted Arrays

**File:** `merge_sorted_arrays.py`

🔍 Merge two sorted arrays into one sorted array.

🔗 LeetCode: https://leetcode.com/problems/merge-sorted-array/

⚙️ Approach:

* Use two pointers
* Compare elements and append the smaller one
* Add remaining elements

⏱️ Time Complexity: `O(n + m)`
💾 Space Complexity: `O(n + m)`
